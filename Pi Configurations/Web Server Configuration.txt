----
Install Apache
----

Source: https://www.raspberrypi.org/documentation/remote-access/web-server/apache.md

In Pi Terminal:

sudo apt-get install apache2 -y

----
Install PHP
----

In Pi Terminal:

sudo apt-get install php5 libapache2-mod-php5 -y

----
Place PHP Files
----

Save all files for web server in:

/var/www/html/
