----
Kodi Installation
----

Source: http://kamilslab.com/2017/02/08/how-to-install-kodi-on-the-raspberry-pi-3-running-raspbian/

In Pi Terminal:

sudo apt-get install kodi

----
Start Kodi
----

In Pi Terminal:

kodi-standalone

----
Install add-ons
----

Within Kodi:

Search for add-ons
Select the add-on required
Install the add-on

Enables use of add-on as a service
